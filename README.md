# 📱 Mobile Sales Dashboard

An interactive Power BI dashboard designed to analyze mobile sales performance across brands, cities, payment methods, customer ratings, and time periods. The dashboard provides actionable business insights through dynamic visualizations, KPIs, and interactive filtering.

---

## 📌 Project Overview

This dashboard helps businesses monitor sales performance, identify top-performing products and regions, understand customer purchasing behavior, and track revenue trends to support data-driven decision-making.

---

## 🎯 Business Objectives

- Track overall sales performance
- Analyze revenue and transaction trends
- Compare brand-wise sales
- Evaluate city-wise performance
- Monitor customer ratings
- Analyze payment method preferences
- Identify sales patterns over time

---

## 🚀 Features

### Dashboard Highlights

- 📊 KPI Cards
  - Total Sales
  - Total Quantity Sold
  - Average Sales
  - Total Transactions

- 📈 Sales Trend Analysis

- 📱 Brand Performance Analysis

- 🌍 City-wise Sales Analysis

- ⭐ Customer Rating Analysis

- 💳 Payment Method Distribution

- 🎛 Interactive Slicers & Filters

- 📅 Time-based Analysis
